Hi, I'm Melody Henrich.

I'm a Northeastern University graduate with a BS in Computer Science and Game Development. Within CS+GAME, I focus on graphics programming and game engine development. I also produce music, study linguistics, and have a (maybe) unhealthy obsession with document markup. I do a lot of projects, and sometimes I finish them.

Projects

Physicalized Game Inerface

Software that lets you use a webcam to detect tags and reproject images onto them in physical space. Intended for augmented tabletop roleplay gaming. Started as an independent study @ Northeastern.

Enigma

An object-oriented interface for OpenGL that I ambitiously labeled an "engine." Includes interfaces for many common OpenGL objects and Blinn-Phong shading. Built as a way to learn the OpenGL API.

9.81

A 2d puzzle-platformer game about manipulating gravity to defeat enemies. Features procedural graphics using shaders to draw the gravity indicator. Made for Rapid Idea Prototyping @ Northeastern.

Dinlih Renderer

A simple HTML/CSS-only site to render romanized Neo-Chakobsa into the Dinlih script created by David J. Peterson for use in the Villeneuve Dune movies.

Musicking on Mars

A one-pager TTRPG designed to encourage improvised music-making. Created as a part of Topics in Game Design @ Northeastern

Lost in Translation

A multiplayer epistolary game about understanding and communication using constructed languages. Created as a part of Topics in Game Design @ Northeastern.

Dodgy Diplomacy

A debate-oriented tabletop roleplay game where players act as devious politicians attempting to deceive each other and the public by intentionally using fallacious logic and misleading speech. Created as a part of Topics in Game Design @ Northeastern.
